APLICACIONES


Función del Formulario:

El formulario mantenimiento aplicaciones es una interfaz que permite gestionar la información relacionada con las aplicaciones dentro del sistema. Este formulario está diseñado para facilitar tareas como la búsqueda, inserción, modificación, actualización y eliminación de datos, proporcionando un control sencillo sobre el registro de aplicaciones.




Funciones del Formulario

El formulario contiene los siguientes botones y opciones que permiten interactuar con los datos de las aplicaciones de manera eficiente:

**1. Buscar por ID:**
Descripción: Este botón permite realizar búsquedas de una aplicación específica mediante el ingreso de su ID.
Campo relacionado: El campo de ID Aplicación es donde se introduce el identificador único de la aplicación que se desea consultar.
Resultado: Al buscar, se mostrarán en pantalla los siguientes datos de la aplicación seleccionada:
- **Nombre de la Aplicación:** Nombre con el cual se identifica la aplicación.
- **Descripción:** Detalle o propósito de la aplicación.
- **Estado:** Determina si la aplicación está activa o inactiva. Esto se controla mediante un Radio Button donde el usuario puede seleccionar una de las dos opciones: **Activo** o **Inactivo**.

**2. Botón Nuevo:**
Descripción: Limpia el formulario y lo deja listo para ingresar una nueva aplicación. Los campos quedan habilitados para ser completados con información nueva.

**3. Botón Ingresar:**
Descripción: Permite registrar una nueva aplicación en el sistema. Al pulsarlo, los datos ingresados (ID, Nombre, Descripción y Estado) se guardan en la base de datos.

**4. Botón Modificar:**
Descripción: Habilita los campos del formulario para modificar los datos de la aplicación seleccionada, permitiendo realizar cambios.

**5. Botón Actualizar:**
Descripción: Al presionar este botón, los cambios realizados a una aplicación existente se guardan en la base de datos, actualizando la información previamente registrada.

**6. Botón Eliminar:**
Descripción: Este botón elimina permanentemente del sistema la aplicación seleccionada, retirando sus datos de la base de datos.

**7. Botón Cancelar:**
Descripción: Permite cancelar cualquier operación en curso. Al presionarlo, se limpian los campos del formulario y se restablece su estado inicial, sin realizar ninguna acción sobre la base de datos.
