CAMBIO DE CONTRASEÑA


Función del Formulario:

1.El formulario de Cambio de Contraseña

2. Para cambiar la contraseña se tiene que ingresar el id del usuario y luego dar click en el boton buscar,al realizarlo si el usuario existe se llenaran todos los campos con los datos del Usuario


3. Luego se ingresa la nueva contraseña y se dara click en el boton guardar, y la contraseña se habra cambiado.
