ASIGNACIÓN DE PERFILES


Función del Formulario:

1. El formulario consta de dos partes, parte de abajo se puede consultar que perfiles tiene asignado el usuario y parte de arriba para hacer la asignación de nuevos perfiles al usuario seleccionado.

2. En la lista desplegable seleccionamos el usuario deseado y nos mostrara los perfiles que tiene asignado y un mensaje del dato seleccionado.


2. Clic en el boton agregar, para agregar una asignación.


3. Le damos clic en buscar para consultar los perfiles de usuarios registrados.


4. También podemos dar clic en cancelar para detener la acción actual. Al igual contamos con el boton quitar, para poder quitar un perfil asignado.
