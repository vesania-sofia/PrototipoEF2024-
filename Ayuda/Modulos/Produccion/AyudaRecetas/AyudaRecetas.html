Funciones de los Botones Para el Apartado de Recetas

  • Botón Agregar: Añade un nuevo registro en la base de datos.
  • Botón Modificar y Actualizar: Permite modificar los registros de la base de datos y tambien cada vez que se modifica se actualizan los datos.
  • Botón Eliminar: Te permite seleccionar un registro para borrarlo de la base de datos.
  • Botón Consultar: Consulta los datos seleccionando el ID en la tabla.
  • Botón Guardar: Guarda los cambios realizados en los registros, ya sea nuevos.
  • Botón Salida: Cierra la ventana de manera segura.

Preguntas Frecuentes

  • ¿Por qué no puedo modificar los datos? Puede que no tengas los permisos necesarios para modificar los registros.
  • ¿Puedo cambiar solo un campo en un registro? Sí, puedes modificar un campo específico, lo que actualizará el registro completo.
  • ¿Por qué no puedo modificar el ID? El ID se genera automáticamente al crear un nuevo registro y no puede ser modificado.
  • ¿Los datos tienen una revisión? Sí, los datos son revisados antes de ser almacenados en la base de datos para asegurar su validez.
  • ¿Como se llenan los campos? Los compos se rellenan conforme lo que se indica en los campos solo tener cuidado de no ingresar letras donde van campos numericos como por ejemplo: Cantidad, Dias y ect.
  • ¿Porque no se puede Eliminar un registro de la tabla Receta detalles? No, No se puede Eliminar un registro de la tabla Receta Detlles porque la opcion no esta permitida.
