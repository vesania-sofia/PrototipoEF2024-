Funciones de los Botones

Botón Agregar: Despliega una ventana para añadir un nuevo registro en la base de datos.

Botón Modificar: Permite abrir una ventana para modificar los datos de un registro existente.

Botón Eliminar: Te permite seleccionar un registro para borrarlo de la base de datos.

Botón Actualizar: Recarga el DataGridView para mostrar los datos más recientes.

Botón Consultar: Abre una ventana que permite realizar búsquedas avanzadas en la base de datos.

Botones Anterior y Siguiente: Te permiten navegar entre los registros uno por uno.

Campos y Selección de Datos

El formulario incluye varios campos y combos que obtienen información específica.

Fecha: Selecciona la fecha en la que se registra la póliza.

Tipo de Póliza: Selecciona el tipo de póliza que se está creando (por ejemplo, "DE AJUSTE").

Concepto: Campo para ingresar una breve descripción o concepto de la póliza.

Detalle de la Póliza

En esta sección, puedes especificar detalles adicionales para la póliza.

Cuenta: Selecciona la cuenta a la cual se aplicará la operación.

Cargo/Abono: Selecciona si la operación es un cargo o un abono.

Valor: Ingresar el valor correspondiente para la operación.

Aceptar: El botón de verificación (check) agrega la cuenta con su respectiva operación a la póliza.

Quitar: Este botón permite eliminar una cuenta específica de la póliza.

Detalle de Movimiento

La tabla en la parte inferior muestra el detalle de los movimientos agregados a la póliza, incluyendo el código, cuenta, cargo y abono. Los totales de cargo y abono se muestran al final del formulario.
